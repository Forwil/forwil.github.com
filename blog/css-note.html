CSS note

Introduction

Those day I begin to read < CSS: The Definitive Guide >, It describe the standard of CSS2.0 and CSS2.1.

In area of web front end, CSS is designed to control the style of HTML element. The first time I use CSS is in junior high school, when I used the blog of baidu.com---hi.baidu.com. It provide users the authority to edit CSS file of their blog. At that time I just know how to adjust a few thing.

After I go to university and major in computer science. I start to learn web develope. First I learn PHP+Mysql, and know how to write a little HTML with CSS. But still can't deal well with CSS style in layout. The CSS seems lose control under my code, Event I already wrote a complete website with Webpy and Bootstrap. I still don't understand how CSS control the layout of website.

The function of CSS is to reduce the coupling of HTML code. Before CSS came out you need to add some attribute that describe style of HTML in basic HTML code, that mixed the "content" and "style" in one material. Sometimes you even have to put away the "content" to abtain some special "style". Here is a example:

<font size="+" face="Helvetical" color="red">
    Page Title
</font>

It mix up the "content" and "style", and is hard for program to know the tag "font" is represent a title.

So we need something beyond HTML to describe the style of HTML element. It must easy to apply on HTML and has more ability to express more visual effect.

Element

There has two type of element of HTML, we treat them in two different way.

replaced element

It means the HTML tag will be replace by some attribute and do not contain the content of the doc. Such as img element, it replace by the image that the "src" refer to:

<img src="howdy.gif">

The "input" element is also a replaced element that we are familiar.

nonreplaced element

The most HTML element are nonreplaced element:

<span>hi there</span>

display

inline element and block is treat different in CSS. Every element has a default display and can be change by "display". Block element can contain a inline element, but inline element can't contain a block element. IF we change the display value into "none", it will disappear .

alernate style sheet

We can add a list of CSS sheet into one page, and let the user choice which one to apply. Throught add rel="alernate stylesheet" and add title in link tag let user know:

<link rel="stylesheet" type="text/css">
    href="sheet1.css" title="Default">
<link rel="alernate stylesheet" type="text/css"/>
    href="bigtext.css" title="Big text">
<link rel="alernate stylesheet" type="text/css"/>
    href="zany.css" title="Crazy colors"/>

selector

class selector

We use "class='x'" to let the element become the class "x", In CSS we can follow below to control the style of class x:

.x{
    style:value;
}

ID selector

We can also add "id='y'" to identity one element, The diffenence between class selector and ID selector is that ID selector can only apply in one ID(defind by XHTML):

#y{
    style:value;
}

attr selector

CSS previde a powerful attr selector. Supose we want to select the element with class, we can follow below:

*[class]{
    style:value;
}

Or even more accurate, we can add "=" to select the element that has given value, such we want to get the link value link to certain url:

*[href="url"{
    style:value;
}

We also has special matching method:

  1. ~=: word match
  2. ^=: begin match
  3. $=: end match
  4. *=: contain match

DOM tree selector

If we want to select the "em" descendant of "h1", we can follow the code below:

h1 em{
    style:value;
}

If we only want to select the "em" children "h1":

h1 > em{
    style:value;
}

Select brother? For example we want to clear the top-distance of the follow paragraph of h1. We can:

h1 + p{
    margin-top: 0;
}

pseudo-class

Pseudo-class is designed to describe the special action of a element, It means if the element be treat in certain way it become the pseudo-class. To defind the style of pseudo-class is like to defind "when something happen, you become ..."

  1. :visited if a link is visited
  2. :focus if it abtain the focus of input
  3. :hover if the mouse stay over
  4. :active some be click by mouse

size

In CSS:

  • 1em it means the lenth of now font's size.
  • 1ex offen means the size of lowercase "x" of now font, maybe 0.5em.
  • 1px means 1pixel

Text

  • text-indent: is the indentation, offen use xem. It can be negative
  • text-align: left | center | right | justify
  • line-height: (line-height - font-size) / 2 is the up and down gap.
  • vertical-align: baseline | sub | super | top | text-top | middle | bottom
  • word-spacing: word betweens
  • letter-spacing: letter betweens
  • text-transform: uppercase | lowercase | capitalize | none
  • text-decoration: underline | overline | line-through | blink
  • text-shadow: color + xoffset + yoffset + r
  • white-space: deal with space merge or not, link break be ignore or not
  • direction: rtl means right to left
